1. Color

Defines the text color of an element.

p { color: red; }

2. Background

Sets the background color, image, or gradient.

div { background: linear-gradient(#4facfe, #00f2fe); }

3. Margin

Controls space outside the element’s border.

div { margin: 20px; }

4. Padding

Controls space inside the element’s border.

div { padding: 15px; }

5. Border

Defines width, style, and color of the border.

div { border: 2px solid white; }

6. Font-size

Specifies the size of the text.

h1 { font-size: 32px; }

7. Flexbox

A layout model for aligning and distributing items.

.container { display: flex; justify-content: center; }

8. Grid

A two-dimensional layout system for web design.

.grid { display: grid; grid-template-columns: 1fr 1fr; }
